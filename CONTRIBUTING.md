# Contributing

Contributions to the [Koopa (COBOL) Parser Generator](README.md) project are very much appreciated !

I especially welcome additions to our testsuite.
So if you have some real life COBOL code lying around, and you're willing and able to share it under a BSD license, please consider doing so.


## Guidelines

* Contributions are welcome in the form of pull requests.
* If you are working on a specific issue and/or want to discuss your changes first, please open an issue in the issue tracker.
* All contributions to the project are expected to be released under the same license as the main project ([BSD](LICENSE.md)).
* For licensing reasons, the project does not accept any code generated by AI tools.
